# Халтарынхан · Ургийн бичиг (Django)

`haltarynkhan-tree-combined.html`-ийн бүх мэдээллийг **өгөгдлийн сан** дээр
шилжүүлж, Django-оор дахин бүтээсэн ургийн модны систем.

## Онцлог
- **200 хүний** мэдээлэл өгөгдлийн сангаас (эх өгөгдөл / initial data) ирдэг.
- Оригинал ургийн модны **загвар, хайлт, zoom, дэлгэрэнгүй** бүрэн хадгалагдсан.
- Шинэ талбарууд: **Утас, Сошиал хаяг, Гэрийн хаяг, Гэр бүлийн зураг**.
- **Django админ** — бүх мэдээллийг удирдана (`/admin/`).
- **Нэвтрэлт**: нэвтрэх нэр = **утасны дугаар**, нууц үг = **төрсөн он-сар-өдөр**.
- **Засах эрх**: хүн бүр өөрийн болон **өөрөөс доош бүх үр удмынхаа** мэдээллийг засна.
- **Урх болсон (гэр бүлтэй) хүн** бүрийн **гэр бүлийн зураг** дэлгэрэнгүй хэсэгт харагдана.

## Ажиллуулах
```bash
cd "family tree"
./venv/bin/python manage.py migrate        # анх удаа: DB + эх өгөгдлийг ачаална
./venv/bin/python manage.py runserver
```
- Ургийн мод:  http://127.0.0.1:8000/
- Нэвтрэх:     http://127.0.0.1:8000/newtreh/
- Админ:       http://127.0.0.1:8000/admin/  (admin / admin12345)

## Нэвтрэлтийг идэвхжүүлэх
Анхны өгөгдөлд утасны дугаар байхгүй тул нэвтрэхийн тулд:
1. `/admin/`-д ороод хүсэн буй хүнийг нээж **Утас** талбарыг бөглөнө.
2. Тухайн хүн **утсаараа** нэвтэрч, нууц үг нь **төрсөн огноо** нь болно
   (ж: `1965.02.26` эсвэл зөвхөн `1965`). Огноог цифрээр нь жишдэг тул
   `19650226`, `1965.02.26` хоёулаа тохирно.

## Бүтэц
- `familytree/` — төслийн тохиргоо (settings, urls)
- `tree/models.py` — `Person` модель (parent FK-тэй мод)
- `tree/auth_backend.py` — утас/төрсөн-огноо нэвтрэлт
- `tree/migrations/0002_load_initial_data.py` — HTML→DB эх өгөгдөл ачаалагч
- `tree/initial_data.json` — оригинал HTML-ээс задалсан 200 хүний өгөгдөл
- `tree/templates/tree/` — `tree.html` (мод), `login.html`, `edit.html`
- `media/family_photos/` — байршуулсан гэр бүлийн зургууд
